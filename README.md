This site is hosted via **GitHub Pages**. To deploy:

1. Push all files to the `main` branch of your `seginstallationtool.github.io` repository.
2. Go to **Settings → Pages** and ensure source is set to `main` branch / root (`/`).
3. Your site will be live at `https://seginstallationtool.github.io` within a few minutes.
